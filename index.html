DNA Electrostatics in R

Fast algorithm of calculation of DNA electrostatics was developed in 1999 by Anatoly Sorokin (@lptolik) and Timur Dzhelyadin. Since then we have looked through regulatory elements of many genomes and show that electrostatical properties of regulatory DNA sites could explain a lot of features of DNA-protein recognition process.

All analysis in that papers was done with Matlab, which is not free. Active development of R and Bioconductor project especially make us to migrate our code to R/Bioconductor, so it will be much more accessible to bioinformatics community.

INSTALLATION

The reldna library is pure R code so it should work on any platform where R is installed. Installation process is really straightforward:

  1. Download latest version of the library
  2. Use standard R installation procedure
    • Chechk that all dependencies are installed in your system
    • Install library R CMD INSTALL reldna_THE_VERSION_YOU_VE_DOWNLOADED.tar.gz News and results of sequence analysis with reldna library are also available at the Laboratory of Cell Genome Functioning Mechanisms site.
